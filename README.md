# 🏨 Hotel Management System (Java + JDBC)

## 📌 Project Description

This is a console-based Hotel Management System developed using Java and JDBC.
It allows users to manage hotel reservations and customer records.

---

## 🚀 Current Features

* Reserve a room
* View reservations
* Get room number
* Update reservations
* Delete reservations

---

## 🛠️ Technologies Used

* Java
* JDBC
* MySQL

---

## ⚙️ How to Run

1. Clone the repository:
     https://github.com/GoyalVarnit07/hotel_management_system-project
2. Open the project in IntelliJ IDEA
3. Configure MySQL database connection
4. Run the `Main.java` file

---

## 📷 Sample Output

Console-based menu system:

1. Reserve a room
2. View Reservations
3. Get Room Number
4. Update Reservations
5. Delete Reservations
6. Exit
